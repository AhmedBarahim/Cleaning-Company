Infrared Therometer Forhead

Description

We know that some thermometer operators can get a wee bit too close for comfort. So get yours today and it comes with an adjustable stand!.

Features
  • °C/°F units.
  • Fever alarm at 38°C / 100.4°F.
  • Accurate up to ±0.2°C / 0.4°F.
  • 0.5 seconds to measure body temperature.
  • Large screen display.
Technical Specifications
  • Infrared accuracy ±0.2°C / 0.4°F.
  • Response time: 0.5s.
  • Measuring distance 5-15cm.
  • Power supply: DC 4.2V-5V.
  • Standby: about 1 week.
  • Net weight: 400g.
  • Package size: 243 x 80 x 195cm.
